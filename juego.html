juego module¶

juego.py — Bucle principal + parser de comandos

Implementa el bucle de entrada/salida por consola, interpreta la línea del usuario y delega la acción al módulo correspondiente, además de gestionar la visualización de mensajes de estado y ayuda

Funciones principales: - interpretar(linea): interpreta la línea de comando y llama a la función adecuada. - iniciar(): inicia el juego y gestiona el bucle principal.

Responsabilidad: - Gestionar el flujo principal del juego. - Interpretar comandos de usuario y delegar acciones. - Mostrar mensajes de estado y ayuda.

Relaciones: - Importa: estado para gestionar el estado del juego. - Importa: movimiento para manejar desplazamientos y descripciones de salas. - Importa: acciones para gestionar el inventario y la interacción con objetos.

juego.iniciar() → None[fuente]¶

Inicia el bucle principal del juego.

Muestra el mensaje de bienvenida, inicializa el estado de la sala inicial y mantiene la ejecución activa hasta que se cumpla la condición de victoria o el usuario decida salir.

Devuelve:

No devuelve ningún valor.

Tipo del valor devuelto:

None

Muestra:

  • Exception – Captura errores inesperados durante la ejecución para informar al usuario.

  • NotImplementedError – Si la funcionalidad solicitada no está implementada.

  • ValueError – Si la entrada es inválida o incompleta.

  • Exception – Si ocurre cualquier otro error inesperado.

juego.interpretar(linea) → str[fuente]¶

Interpreta el comando introducido por el usuario y ejecuta la acción correspondiente.

Divide la cadena de entrada y determina mediante el primer token qué función del motor de juego debe invocarse.

Parámetros:

linea (str) – La línea de comando completa introducida por el usuario.

Devuelve:

El mensaje de respuesta generado tras la acción o un código de salida.

Tipo del valor devuelto:

str
